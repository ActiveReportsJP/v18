CustomTileProviderのサンプルを実行するためにはTestDesignerProプロジェクトまたはTestViewerプロジェクトをスタートアッププロジェクトに設定している必要があります。
デフォルトではTestDesignerProプロジェクトがスタートアッププロジェクトに設定されています。

TestViewerプロジェクトをスタートアッププロジェクトに変更する手順は以下の通りです。
1. VisualStudioにて"CustomDataProvider.sln"を実行します。
2. ソリューションエクスプローラーより、TestViewerプロジェクトを選択し、右クリックメニューを開きます。
3. [スタートアッププロジェクトに設定]を選択します。
4. サンプルを実行します。
